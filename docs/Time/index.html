Structure Time 

struct Time

A struct representing a time value with hour, minute, second, and day type. The struct provides computed properties for formatted time display and adjusts minutes and seconds when their values exceed 59.

Properties

hour 

var hour: Int

The hour component of the time (0-23).

min 

var min: Int

The minute component of the time (0-59).

When set, if the minute value exceeds 59, the minute is reset to 0 and the hour is incremented by 1.

sec 

var sec: Int

The second component of the time (0-59).

When set, if the second value exceeds 59, the second is reset to 0 and the minute is incremented by 1.

day​Type 

var dayType: String

A string representing the type of day (e.g., "AM" or "PM").

time 

var time: String

A computed property that returns the formatted time as a string.

The formatted time string is in the format: "HH:MM:SS AM/PM" or "HH:MM:SS" depending on the day type.

  • If hour, minute, or second is less than 10, it will be prefixed with a zero.

Example:

  • "05:09:03 PM"

  • "10:05:02 AM"
